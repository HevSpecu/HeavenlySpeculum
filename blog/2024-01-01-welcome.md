## 关于这个项目

这个项目展示了如何将 Docusaurus 的强大功能与 Fumadocs 的精美设计相结合。

### 主要特性

- ✅ 完整的 Tailwind CSS 集成
- ✅ Fumadocs 设计系统
- ✅ 深色模式支持
- ✅ 响应式设计
- ✅ 现代化的 UI 组件

## 代码示例

这里有一个简单的 TypeScript 示例：

```typescript
interface Config {
  title: string;
  description: string;
}

const config: Config = {
  title: 'My Documentation',
  description: 'A beautiful docs site'
};
```

## 开始使用

要开始使用这个主题，只需克隆仓库并运行：

```bash
npm install
npm start
```

享受创建美观文档的乐趣！

:::tip 专业提示

不要忘记自定义 `docusaurus.config.js` 中的配置以匹配您的项目需求。

:::
